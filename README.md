# UNIX Task Manager Project

## Description
This project is a simple task manager web application developed for the UNIX course.

The application contains:
- HTML frontend
- CSS styling
- JavaScript
- PHP backend
- MariaDB database configuration
- Docker container setup

## Containers
The project uses 3 containers:
1. Web server container (PHP + Apache)
2. Database container (MariaDB)
3. CDN container (Nginx)

## Features
- Display tasks
- Add task page
- JSON data handling
- Docker infrastructure

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MariaDB
